# Medical Insurance Cost Prediction (Multiple Linear Regression)

## 📌 Project Overview

This project uses **Multiple Linear Regression** to predict medical insurance charges based on patient information.

The dataset contains attributes such as:

* Age
* Sex
* BMI
* Number of children
* Smoking status
* Region
* Medical charges

The goal is to build a regression model that estimates the **insurance cost** using these features.

---

## ⚙️ Workflow

1. Load dataset
2. Data preprocessing
3. Handle categorical variables
4. Feature selection
5. Train Multiple Linear Regression model
6. Evaluate model performance
7. Predict medical charges
